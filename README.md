# Court Click Dashboard

A responsive dashboard application built using Next.js, TypeScript, and Ant Design.

## Features

* Dashboard statistics cards
* User management table
* Search functionality
* Status tags
* View details modal
* Assign clerk modal
* Tags management
* Filter functionality
* Responsive design
* TypeScript support

## Technologies Used

* Next.js 16
* React 19
* TypeScript
* Ant Design

## Installation

```bash
npm install
npm run dev
```

Open:

http://localhost:3000

## Production Build

```bash
npm run build
npm start
```
